# ElectroMentor

ElectroMentor is a fully offline gamified Electronics Engineering learning app built using Flutter.

## Features

- 5 Electronics subjects
- 25 modules
- 125 lessons
- XP & Level system
- Progress tracking
- Offline search
- Smart mentor system
- Dark premium UI
- No backend
- No API
- No ads

## Subjects

- Basic Electronics
- Digital Logic
- Instrumentation
- PLC
- Robotics
